# BETH Stack Product List

To install dependencies:

```bash
bun install
```

Required environment variables:

```bash
DATABASE_AUTH_TOKEN=...
DATABASE_URL=...
```

To run:

```bash
bun run dev
```

This project was created using `bun init` in bun v0.6.14. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
